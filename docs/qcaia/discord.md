# Using QCAIA on Discord

On Discord, QCAIA works through **slash commands**. Type `/`, pick a command, fill in your question, and QCAIA replies right in the channel.

## Join the Discord

Join the official QoreChain Discord server:

**[discord.gg/qorechain](https://discord.gg/qorechain)**

Once you're in, head to the **#general** channel to talk to QCAIA.

## Slash commands

QCAIA responds to four slash commands:

| Command | What it does |
| --- | --- |
| **`/ask <question>`** | Ask QCAIA anything about QoreChain. |
| **`/about`** | Learn what QoreChain is and what QCAIA can do. |
| **`/tokenomics`** | Get a summary of the QOR token and its economics. |
| **`/contact`** | Find official ways to get in touch with the team. |

## Asking a question

Use **`/ask`** in the **#general** channel:

```
/ask How does QoreChain's bridge work?
```

QCAIA reads your question and replies. Responses come back as **Discord embeds** — formatted message cards that keep the answer clean and easy to read. You can ask follow-up questions the same way.

Because QCAIA is **multi-language**, you can also write your `/ask` question in another language; it will translate, answer, and reply. See [Examples](/qcaia/examples) for a demonstration.

## Tips

- Keep each `/ask` focused on one question for the clearest answer.
- Use **`/about`** if you're new and want a quick orientation.
- Use **`/tokenomics`** for a fast QOR summary before digging deeper with `/ask`.
- QCAIA is in **beta** — if an answer isn't quite right, try rephrasing your question.

For the mention-based Telegram experience instead, see [Using QCAIA on Telegram](/qcaia/telegram).
