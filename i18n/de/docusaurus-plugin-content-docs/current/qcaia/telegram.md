# QCAIA auf Telegram nutzen

Auf Telegram ist QCAIA **erwähnungsbasiert** — es gibt keine Slash-Befehle. Sie stellen einfach Ihre Frage ganz natürlich, erwähnen den Bot (oder QoreChain), und er antwortet.

## Der Gruppe beitreten

Treten Sie der offiziellen QoreChain-Community-Gruppe bei:

**[t.me/QoreChainCommunity](https://t.me/QoreChainCommunity)**

QCAIA ist **nur in Gruppen** verfügbar und unterstützt Telegram-**Forum-Themen**, sodass er innerhalb der organisierten Themen-Threads der Gruppe funktioniert.

## Wie man mit QCAIA spricht

QCAIA antwortet, wenn Sie eine der folgenden Aktionen ausführen:

1. **Ihn direkt erwähnen** — fügen Sie **`@QCAIA_Bot`** in Ihre Nachricht ein.
2. **Auf eine seiner Nachrichten antworten** — beantworten Sie eine von QCAIA gesendete Nachricht oder fragen Sie nach, und er wird reagieren.
3. **„QoreChain" erwähnen** in Ihrer Nachricht im Thema **General** — QCAIA nimmt es auf und antwortet.

In allen Fällen schreiben Sie einfach Ihre Frage so, wie Sie es normalerweise tun würden.

### Beispiele

Den Bot direkt erwähnen:

```
@QCAIA_Bot How does QoreChain's bridge work?
```

Im Thema **General** QoreChain erwähnen:

```
What is the QOR fee distribution on QoreChain?
```

Wenn Sie auf eine QCAIA-Nachricht antworten, wird das Gespräch einfach fortgesetzt — keine Erwähnung erforderlich.

## Tipps

- In anderen Themen-Threads als **General** erwähnen Sie **`@QCAIA_Bot`** (oder antworten Sie ihm), um sicherzustellen, dass QCAIA Ihre Frage sieht.
- QCAIA ist **mehrsprachig**: Fragen Sie in Ihrer eigenen Sprache, und er wird übersetzen, beantworten und antworten. Siehe [Beispiele](/qcaia/examples).
- Halten Sie jede Frage fokussiert, um die klarste Antwort zu erhalten.
- QCAIA befindet sich in der **Beta**-Phase — wenn eine Antwort nicht ganz richtig ist, formulieren Sie sie um und versuchen Sie es erneut.

Bevorzugen Sie Slash-Befehle? Siehe [QCAIA auf Discord nutzen](/qcaia/discord).
