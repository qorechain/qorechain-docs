# Utiliser QCAIA sur Discord

Sur Discord, QCAIA fonctionne via des **commandes slash**. Tapez `/`, choisissez une commande, saisissez votre question, et QCAIA répond directement dans le salon.

## Rejoindre le Discord

Rejoignez le serveur Discord officiel de QoreChain :

**[discord.gg/qorechain](https://discord.gg/qorechain)**

Une fois à l'intérieur, rendez-vous dans le salon **#general** pour discuter avec QCAIA.

## Commandes slash

QCAIA répond à quatre commandes slash :

| Commande | Ce qu'elle fait |
| --- | --- |
| **`/ask <question>`** | Posez à QCAIA n'importe quelle question sur QoreChain. |
| **`/about`** | Découvrez ce qu'est QoreChain et ce que QCAIA peut faire. |
| **`/tokenomics`** | Obtenez un résumé du token QOR et de son économie. |
| **`/contact`** | Trouvez les moyens officiels de contacter l'équipe. |

## Poser une question

Utilisez **`/ask`** dans le salon **#general** :

```
/ask How does QoreChain's bridge work?
```

QCAIA lit votre question et répond. Les réponses reviennent sous forme d'**embeds Discord** — des cartes de message formatées qui gardent la réponse claire et facile à lire. Vous pouvez poser des questions de suivi de la même manière.

Comme QCAIA est **multilingue**, vous pouvez aussi écrire votre question `/ask` dans une autre langue ; il traduira, répondra et vous répondra. Voir [Exemples](/qcaia/examples) pour une démonstration.

## Conseils

- Gardez chaque `/ask` centré sur une seule question pour obtenir la réponse la plus claire.
- Utilisez **`/about`** si vous débutez et souhaitez une orientation rapide.
- Utilisez **`/tokenomics`** pour un résumé rapide de QOR avant d'approfondir avec `/ask`.
- QCAIA est en **bêta** — si une réponse n'est pas tout à fait juste, essayez de reformuler votre question.

Pour l'expérience Telegram basée sur les mentions, voir [Utiliser QCAIA sur Telegram](/qcaia/telegram).
