# Exemples QCAIA

Voici quelques exemples concrets de questions posées à QCAIA et le type de réponse qu'il fournit. Les questions utilisent la syntaxe `/ask` de Discord ; sur Telegram, vous poseriez la même question en mentionnant **`@QCAIA_Bot`** ou QoreChain (voir [Telegram](/qcaia/telegram)).

Ces réponses sont **illustratives** — de courtes paraphrases en langage clair de la documentation publique de QoreChain. La formulation réelle de QCAIA variera, et il est en **bêta**.

## Exemple 1 — Le pont

**Vous demandez :**

```
/ask How does QoreChain's bridge work?
```

**QCAIA répond :**

> Le pont de QoreChain vous permet de déplacer des actifs entre QoreChain et d'autres réseaux. Il combine les transferts QCB avec IBC pour la connectivité inter-chaînes, le support IBC étant déployé d'abord sur le testnet. Pour la conception complète, consultez la documentation du pont.

_En savoir plus : [Architecture du pont](/architecture/bridge-architecture)._

## Exemple 2 — PRISM

**Vous demandez :**

```
/ask What is PRISM?
```

**QCAIA répond :**

> PRISM est la couche d'optimisation par apprentissage par renforcement de QoreChain. Elle ajuste en continu le réseau pour la performance et l'efficacité à mesure que les conditions évoluent. Vous pouvez lire l'analyse approfondie dans la documentation PRISM.

_En savoir plus : [Moteur de consensus PRISM](/architecture/prism-consensus-engine)._

## Exemple 3 — Distribution des frais

**Vous demandez :**

```
/ask What is the QOR fee distribution?
```

**QCAIA répond :**

> Les frais de transaction sur QoreChain sont répartis entre plusieurs destinations : 37 % / 30 % / 20 % / 10 % / 3 %. Cela répartit la valeur entre les personnes et les fonctions qui maintiennent le réseau en activité. Consultez la documentation sur la tokenomics pour le détail de chaque part.

## Exemple 4 — Multilingue (espagnol)

QCAIA traduit automatiquement les questions non anglophones, y répond et renvoie la réponse. La voici en espagnol :

**Vous demandez :**

```
/ask ¿Qué es QoreChain?
```

**QCAIA répond :**

> QoreChain es una blockchain de capa 1 con seguridad post-cuántica, un motor de IA (QCAI) y el motor de consenso PRISM. Está diseñada para ser rápida, segura y preparada para el futuro.

Cela fonctionne de la même façon sur Telegram — écrivez simplement votre question dans votre langue en mentionnant **`@QCAIA_Bot`** ou QoreChain.

## Exemple 5 — Orientation rapide

**Vous demandez :**

```
/about
```

**QCAIA répond :**

> Je suis QCAIA, l'assistant IA de QoreChain (bêta). Posez-moi des questions sur la technologie, la tokenomics, l'équipe et l'écosystème de QoreChain sur Discord ou Telegram. Essayez `/ask` suivi de votre question, ou utilisez `/tokenomics` pour un résumé rapide de QOR.
