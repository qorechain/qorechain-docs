# 스마트 컨트랙트 생성기

**스마트 컨트랙트 생성기**는 **QCAI**를 기반으로 일반 언어 설명으로부터 스마트 컨트랙트를 생성합니다. 원하는 내용을 설명하고 대상 블록체인을 선택하면 QCAI가 컨트랙트를 작성해 줍니다. AI 도구를 위한 **17개 블록체인**을 지원하므로, 구축하려는 생태계를 대상으로 지정할 수 있습니다.

지갑을 연결하면 생성한 컨트랙트를 저장하고 관리할 수 있습니다 — [개요 및 시작하기](/dashboard/overview#connect-your-wallet)를 참고하세요.

## 컨트랙트 생성하기

1. **컨트랙트를 설명하세요.** 프롬프트 입력란에 컨트랙트가 수행해야 할 내용을 작성합니다 — 예를 들어 고정 공급량을 가진 토큰, NFT 컬렉션, 또는 베스팅 일정 등입니다. 구체적으로 작성할수록 결과가 좋아집니다.
2. **블록체인을 선택하세요.** 지원되는 블록체인 중에서 대상을 선택합니다. 선택한 항목에 대한 컨트랙트 언어와 카테고리가 선택기 옆에 표시됩니다.
3. **컨트랙트 유형을 선택하세요**(선택 사항). 토큰, NFT, 거버넌스 컨트랙트와 같은 시작 템플릿을 선택하여 생성을 안내합니다.
4. **생성하세요.** **Generate**를 선택합니다. QCAI가 컨트랙트를 생성하는 동안 진행 표시기가 상태를 보여줍니다.

## 결과 검토하기

생성이 완료되면 대시보드는 구문 강조가 적용된 보기로 컨트랙트를 표시하며, 컨트랙트 이름, 블록체인, 언어, 파일 크기, 생성 시간과 같은 세부 정보를 함께 보여줍니다. 사용한 프롬프트도 참고용으로 결과와 함께 표시됩니다.

여기에서 다음을 할 수 있습니다.

- 컨트랙트 코드를 클립보드로 **복사**.
- 선택한 블록체인에 맞는 형식의 파일로 컨트랙트 **다운로드**.
- 컨트랙트를 더 다듬기 위해 **편집**.

## 공유 및 재사용

생성된 각 컨트랙트에는 열거나 공유할 수 있는 전용 페이지가 있습니다. 소유하지 않은 컨트랙트를 열면, 자신의 사본을 만들어 이어갈 수 있도록 **포크**할 수 있습니다.

:::tip 항상 검토하고 테스트하세요
QCAI가 생성한 코드는 훌륭한 출발점일 뿐, 검토를 대체하지는 않습니다. 가치 있는 무언가를 배포하기 전에 컨트랙트를 읽고, [테스트넷](/getting-started/connecting-to-testnet)에서 테스트하며, [컨트랙트 감사기](/dashboard/contract-auditor)를 통해 검사하세요.
:::
