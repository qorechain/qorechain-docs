# 퀘스트

**퀘스트**는 완료하는 동안 QoreChain 사용법을 단계별로 안내하는 가이드형 작업입니다. 각 퀘스트는 지갑에 자금을 충전하거나, 송금을 하거나, 스왑을 해보는 것과 같은 하나 이상의 단계를 묶어 구성되며, 완료하면 보상을 제공합니다. 퀘스트는 네트워크의 기능을 하나씩 직접 체험하며 배울 수 있는 실습 방식입니다.

진행 상황이 추적되도록 지갑을 연결하세요 — [개요 및 시작하기](/dashboard/overview#connect-your-wallet)를 참고하세요.

## 퀘스트 둘러보기

퀘스트 페이지는 이용 가능한 퀘스트 수, 진행 중인 퀘스트 수, 완료한 퀘스트 수를 간단히 요약하며 시작됩니다. **카테고리 탭**(예: 온보딩, 트레이딩, 소셜, 커뮤니티, 학습, 일일, 주간)을 통해 목록을 필터링할 수 있습니다.

각 퀘스트는 다음 정보를 보여주는 카드로 표시됩니다.

- 아이콘, 제목, 짧은 설명.
- **난이도** 배지(예: 쉬움, 보통, 어려움).
- 예상 완료 시간.
- 획득하는 **보상**.
- **상태**: 완료, 진행 중, 시작됨, 잠김.

잠긴 퀘스트에는 해당 퀘스트가 잠금 해제되기 전에 먼저 완료해야 하는 퀘스트가 표시됩니다.

## 퀘스트 완료하기

1. 퀘스트를 열어 전체 작업 목록이 담긴 상세 페이지를 확인합니다.
2. 각 작업을 차례로 진행합니다. 작업에 따라 관련 대시보드 페이지로 연결되어 동작을 수행하게 하거나, 검증할 수 있도록 단계를 확인하도록 요청할 수 있습니다.
3. 진행률 표시줄이 완료한 작업 수를 추적합니다.
4. 모든 작업이 완료되면 **Check Completion**을 선택하여 퀘스트를 검증하고 보상을 받습니다.

일부 퀘스트는 일일 또는 주간 일정에 따라 초기화됩니다. 이러한 퀘스트의 경우 상세 페이지에 초기화 주기와 현재 연속 기록(스트릭)이 표시되므로, 정기적으로 돌아와서 진행을 이어갈 수 있습니다.
