# 지갑

**지갑** 페이지에서 잔액을 확인하고, QOR를 보내거나 받고, 주소를 관리할 수 있습니다. QoreChain 계정은 양자 내성 암호화로 보호되며, 모든 주소는 `qor` bech32 접두사(`qor1...`)를 사용합니다.

먼저 지갑을 연결하세요. [개요 및 시작하기](/dashboard/overview#connect-your-wallet)를 참조하세요.

## 페이지에 표시되는 내용

- 지갑 레이블과 활성 주소를 단축된 형태로 표시하며, 원클릭 복사 버튼을 제공합니다.
- QOR 단위의 **총 잔액**.
- 양자 내성 암호화와 연결된 네트워크를 알려주는 보안 패널.
- 새로고침 컨트롤이 있는 마지막 업데이트 표시기.
- 보유 자산과 거래 내역을 보여주는 **자산** 및 **활동** 탭.

언제든지 새로고침 컨트롤을 사용하여 체인에서 현재 잔액과 최신 활동을 가져올 수 있습니다.

## QOR 보내기

1. **보내기**를 선택합니다.
2. 받는 사람의 주소(`qor1...`)를 입력합니다.
3. 금액과 선택적 메모를 입력합니다.
4. 세부 정보와 예상 수수료를 검토한 후 확인합니다.

받는 사람을 입력하면 실수를 방지할 수 있도록 저장된 연락처와 최근 주소가 제안됩니다. 전송이 제출되면 트랜잭션 해시가 포함된 확인을 받게 되며, 이는 [익스플로러](/dashboard/explorer)에서 열어볼 수 있습니다.

:::caution 주소를 다시 확인하세요
블록체인 전송은 되돌릴 수 없습니다. 보내기 전에 항상 받는 사람의 주소를 확인하세요.
:::

## QOR 받기

1. **받기**를 선택합니다.
2. 보내는 사람과 주소 또는 QR 코드를 공유하거나, 원클릭으로 주소를 복사합니다.
3. 선택적으로 요청 금액과 메모를 입력하여 결제 링크와 다운로드 가능한 QR 코드를 생성합니다.

## 지갑 관리

**내 지갑**을 선택하여 주소 목록을 엽니다. 거기에서 지갑 간 전환, 새 지갑 생성, 기존 지갑 가져오기, 더 이상 필요하지 않은 지갑 삭제를 할 수 있습니다. 활성 지갑은 대시보드 전반에서 보내기, 스왑, 스테이킹 및 기타 서명된 작업에 사용되는 지갑입니다.
