# 포셋

**포셋(Faucet)**은 가치 있는 것을 소비하지 않고 대시보드를 사용해 볼 수 있도록 무료 테스트 토큰을 제공합니다. 이는 **테스트넷 전용** 도구로 — 테스트넷(`qorechain-diana`)에서 사용하여 전송, 스왑, 스테이킹, 컨트랙트 배포를 테스트하기 전에 주소에 자금을 충전하세요.

## 테스트 토큰 요청

1. **테스트넷**에 연결하고 **포셋(Faucet)**을 엽니다.
2. 자금을 충전할 주소(`qor1...`)를 입력합니다. 지갑이 연결되어 있으면 **내 주소 사용(Use my address)**을 선택하여 자동으로 입력합니다. 양식은 계속 진행하기 전에 주소가 유효한지 확인합니다.
3. 요청 버튼을 선택합니다. 자금 충전은 몇 초 안에 처리됩니다.

요청이 성공하면 확인 카드에 전송된 금액과 트랜잭션 해시가 복사 버튼 및 [익스플로러](/dashboard/explorer)에서 트랜잭션을 여는 링크와 함께 표시됩니다.

## 한도

각 주소는 쿨다운 기간당 한 번 포셋에서 요청할 수 있습니다. 페이지는 요청당 정확한 금액과 다시 청구할 수 있게 되기까지의 쿨다운을 보여줍니다. 너무 빨리 다시 요청하면 포셋이 다음에 언제 자격이 되는지 알려줍니다.

## 테스트 토큰으로 할 수 있는 일

테스트 토큰을 사용하면 테스트넷에서 네트워크를 처음부터 끝까지 시험해 볼 수 있습니다:

- [지갑](/dashboard/wallet) 페이지에서 송금하고 수신합니다.
- AMM에서 [스왑](/dashboard/trade)을 시도합니다.
- 검증자에게 [위임](/dashboard/staking-and-validators)합니다.
- 메인넷으로 이동하기 전에 컨트랙트를 배포하고 테스트합니다.

:::note 테스트 가치 전용
포셋 토큰은 테스트넷에 존재하며 실제 가치가 없습니다. 프로덕션 사용을 위한 준비가 되면 메인넷(`qorechain-vladi`)으로 전환하세요.
:::
