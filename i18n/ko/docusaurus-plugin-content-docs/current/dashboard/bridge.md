# 브리지

**브리지(Bridge)**를 사용하면 단일 화면에서 QoreChain과 다른 체인 간에 자산을 이동할 수 있습니다. 모든 브리지 작업은 포스트 양자 암호화로 보호됩니다. 크로스체인 전송의 설계에 대해서는 [브리지 아키텍처](/architecture/bridge-architecture)를 참조하세요.

:::caution 한정 상태
크로스체인 브리지는 현재 테스트넷에 있으며 점진적으로 출시되고 있습니다 — 아직 프로덕션 메인넷 시스템이 아닙니다. 사용 가능한 경로를 보장된 라이브 연결이 아니라 진행 중인 작업으로 간주하고, 테스트넷에서 시작하세요.
:::

브리지를 사용하려면 지갑을 연결하세요 — [개요 및 시작하기](/dashboard/overview#connect-your-wallet)를 참조하세요.

## 자산을 브리지하는 방법

1. 상단 선택기에서 **소스(source)** 체인과 토큰을 선택합니다. 선택기는 토큰, 해당 네트워크, 그리고 잔액을 표시합니다.
2. 하단 선택기에서 **목적지(destination)** 체인과 토큰을 선택합니다.
3. 전송할 금액을 입력합니다. 받게 될 금액은 목적지 측에 표시됩니다.
4. 자산을 본인이 아닌 다른 주소로 보내려면 **다른 주소로 보내기(Send to another)**를 켜고 수신자를 입력합니다.
5. 하단에 표시된 **수수료(fee)**와 결제까지의 **예상 시간(estimated time)**을 검토합니다.
6. 지갑에서 전송을 확인합니다.

두 선택기 사이의 스왑 컨트롤을 사용하면 한 번의 탭으로 소스와 목적지를 서로 바꿀 수 있습니다.

## 팁

- 제출하기 전에 두 체인과 목적지 주소를 모두 확인하세요 — 크로스체인 전송은 되돌릴 수 없습니다.
- 결제 시간은 경로에 따라 다릅니다. 체인과 금액을 변경하면 예상 시간이 업데이트됩니다.
- 전송이 체인 간에 검증되는 방식에 대한 배경 정보는 [자산 브리징](/user-guide/bridging-assets)과 [브리지 아키텍처](/architecture/bridge-architecture)를 참조하세요.
