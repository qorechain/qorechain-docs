# Discord에서 QCAIA 사용하기

Discord에서 QCAIA는 **슬래시 명령**을 통해 동작합니다. `/`를 입력하고, 명령을 선택한 뒤, 질문을 채워 넣으면 QCAIA가 바로 채널에서 답변합니다.

## Discord 참여하기

공식 QoreChain Discord 서버에 참여하세요:

**[discord.gg/qorechain](https://discord.gg/qorechain)**

들어오면 **#general** 채널로 이동하여 QCAIA와 대화하세요.

## 슬래시 명령

QCAIA는 네 가지 슬래시 명령에 응답합니다:

| 명령 | 하는 일 |
| --- | --- |
| **`/ask <question>`** | QoreChain에 관해 무엇이든 QCAIA에게 물어보세요. |
| **`/about`** | QoreChain이 무엇이고 QCAIA가 무엇을 할 수 있는지 알아보세요. |
| **`/tokenomics`** | QOR 토큰과 그 이코노믹스의 요약을 확인하세요. |
| **`/contact`** | 팀에 연락하는 공식 방법을 찾아보세요. |

## 질문하기

**#general** 채널에서 **`/ask`**를 사용하세요:

```
/ask How does QoreChain's bridge work?
```

QCAIA가 질문을 읽고 답변합니다. 응답은 **Discord 임베드** — 답변을 깔끔하고 읽기 쉽게 유지하는 서식 있는 메시지 카드 — 로 돌아옵니다. 같은 방식으로 후속 질문을 할 수 있습니다.

QCAIA는 **다국어**를 지원하므로, `/ask` 질문을 다른 언어로 작성할 수도 있습니다. QCAIA가 번역하고, 답변하며, 응답합니다. 시연은 [예시](/qcaia/examples)를 참고하세요.

## 팁

- 가장 명확한 답변을 위해 각 `/ask`는 하나의 질문에 집중하세요.
- 처음이고 빠른 안내를 원한다면 **`/about`**을 사용하세요.
- `/ask`로 더 깊이 파고들기 전에 빠른 QOR 요약을 보려면 **`/tokenomics`**를 사용하세요.
- QCAIA는 **베타** 상태입니다 — 답변이 딱 맞지 않으면 질문을 다르게 표현해 보세요.

대신 멘션 기반 Telegram 경험을 원한다면 [Telegram에서 QCAIA 사용하기](/qcaia/telegram)를 참고하세요.
