# Telegram에서 QCAIA 사용하기

Telegram에서 QCAIA는 **멘션 기반**입니다 — 슬래시 명령어가 없습니다. 봇(또는 QoreChain)을 멘션하면서 자연스럽게 질문하기만 하면 회신합니다.

## 그룹 참여하기

공식 QoreChain 커뮤니티 그룹에 참여하세요:

**[t.me/QoreChainCommunity](https://t.me/QoreChainCommunity)**

QCAIA는 **그룹 전용**이며 Telegram **포럼 토픽**을 지원하므로, 그룹의 정리된 토픽 스레드 내에서 작동합니다.

## QCAIA와 대화하는 방법

QCAIA는 다음 중 하나를 수행할 때 회신합니다:

1. **직접 멘션** — 메시지에 **`@QCAIA_Bot`** 을 포함합니다.
2. **QCAIA 메시지에 답글** — QCAIA가 보낸 메시지에 답하거나 후속으로 이어가면 응답합니다.
3. **`General` 토픽에서 메시지에 "QoreChain" 언급** — QCAIA가 이를 감지하여 답합니다.

모든 경우에 평소처럼 질문을 작성하기만 하면 됩니다.

### 예시

봇을 직접 멘션하기:

```
@QCAIA_Bot How does QoreChain's bridge work?
```

**General** 토픽에서 QoreChain을 언급하기:

```
What is the QOR fee distribution on QoreChain?
```

QCAIA 메시지에 답글을 다는 것은 단순히 대화를 이어갑니다 — 멘션이 필요 없습니다.

## 팁

- **General** 이외의 토픽 스레드에서는 QCAIA가 질문을 확실히 보도록 **`@QCAIA_Bot`** 을 멘션(또는 답글)하세요.
- QCAIA는 **다국어**입니다: 자신의 언어로 질문하면 번역하고, 답변한 후 회신합니다. [예시](/qcaia/examples)를 참조하세요.
- 가장 명확한 답변을 위해 각 질문은 집중적으로 유지하세요.
- QCAIA는 **베타** 상태입니다 — 답변이 정확하지 않으면 다시 표현하여 시도하세요.
