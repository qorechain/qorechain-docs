# عمليات الرموز

يغطي هذا الدليل رمز QOR، وكيفية إرسال الرموز واستلامها، والاستعلام عن الأرصدة، وفهم نموذج توزيع الرسوم في QoreChain.

:::note
تستخدم الأوامر أدناه شبكة الاختبار **`qorechain-diana`** (معرّف سلسلة EVM رقم **9800**). الشبكة الرئيسية (**`qorechain-vladi`**، معرّف سلسلة EVM رقم **9801**) أصبحت فعّالة منذ 7 يونيو 2026 وتعمل بإصدار السلسلة **v3.1.82** — استبدل معرّف السلسلة ونقاط النهاية الخاصة بالشبكة الرئيسية من صفحة **Connecting to Mainnet** عند إجراء المعاملات على الشبكة الرئيسية.
:::

## معلومات الرمز

| الخاصية                  | القيمة                        |
| ------------------------ | ----------------------------- |
| **تسمية العرض** | QOR                           |
| **التسمية الأساسية**    | uqor                          |
| **التحويل**           | 1 QOR = 1,000,000 uqor (10^6) |
| **معرّف السلسلة**             | `qorechain-vladi` (الشبكة الرئيسية) / `qorechain-diana` (شبكة الاختبار) |
| **بادئة Bech32**        | `qor` (مثال: `qor1abc...xyz`) |

تُحدَّد جميع المبالغ على السلسلة بوحدة **uqor**. عند إرسال المعاملات، حدّد المبالغ دائمًا بوحدة uqor.

## إرسال الرموز

لتحويل رموز QOR من حساب إلى آخر:

```bash
qorechaind tx bank send <from_address> <to_address> <amount>uqor \
  --chain-id qorechain-diana \
  --fees 500uqor
```

**مثال:** إرسال 5 QOR (5,000,000 uqor) إلى عنوان آخر:

```bash
qorechaind tx bank send qor1sender... qor1recipient... 5000000uqor \
  --chain-id qorechain-diana \
  --fees 500uqor
```

يمكنك أيضًا استخدام اسم مفتاح بدلًا من عنوان خام للمرسِل:

```bash
qorechaind tx bank send mykey qor1recipient... 5000000uqor \
  --chain-id qorechain-diana \
  --fees 500uqor
```

## الاستعلام عن الأرصدة

تحقق من رصيد أي حساب:

```bash
qorechaind query bank balances <address>
```

**مثال:**

```bash
qorechaind query bank balances qor1abc...xyz
```

**نموذج للمخرجات:**

```yaml
balances:
- amount: "15000000"
  denom: uqor
pagination:
  next_key: null
  total: "0"
```

يشير هذا إلى أن الحساب يحتفظ بـ 15 QOR (15,000,000 uqor).

## هيكل الرسوم

تُوزَّع رسوم المعاملات في QoreChain على خمس وجهات لمواءمة حوافز الشبكة:

| الوجهة          | الحصة | الغرض                                                         |
| --------------- | ----- | --------------------------------------------------------------- |
| **المُدقِّقون**  | 37%   | يكافئ منتجي الكتل ويؤمّن الشبكة                 |
| **المحروقة**      | 30%   | تُزال نهائيًا من المعروض، مما يولّد ضغطًا انكماشيًا |
| **الخزينة**    | 20%   | تموّل تطوير البروتوكول ومِنح النظام البيئي                |
| **المُخزِّنون**     | 10%   | تُوزَّع تناسبيًا على جميع المُفوِّضين                    |
| **العُقَد الخفيفة** | 3%    | تكافئ مشغّلي العُقَد الخفيفة الذين يقدّمون بيانات الشبكة            |

## قنوات الحرق

تطبّق QoreChain آلية حرق متعددة القنوات. تُزال رموز QOR نهائيًا من التداول عبر 10 قنوات مميزة:

| القناة              | الوصف                                                         |
| -------------------- | ------------------------------------------------------------------- |
| `tx_fee`             | جزء الحرق البالغ 30% من كل رسوم معاملة                       |
| `governance_penalty` | يُحرق عندما تفشل مقترحات الحوكمة في بلوغ النصاب أو يتم نقضها |
| `slashing_burn`      | الجزء المحروق من حصص المُدقِّقين المخصومة                         |
| `bridge_fee`         | الرسوم المحروقة على تحويلات الجسر عبر السلاسل                        |
| `spam_deterrent`     | حرق إضافي يُطبَّق على معاملات البريد العشوائي المُعلَّمة                |
| `epoch_excess`       | الانبعاثات الزائدة عن الهدف التي تُحرق عند حدود الحقبة           |
| `manual_burn`        | عمليات حرق رموز يبدأها المجتمع عبر مقترح حوكمة               |
| `contract_callback`  | الرسوم المحروقة على عمليات استدعاء العقود الذكية الراجعة (callback)                   |
| `cross_vm_fee`       | تُحرق عند تنفيذ استدعاءات عبر الأجهزة الافتراضية (مثل EVM إلى CosmWasm)        |
| `rollup_create`      | 1% من الحد الأدنى للحصة يُحرق عند نشر تجميع (rollup) جديد          |

يمكنك الاستعلام عن إجمالي المبلغ المحروق عبر جميع القنوات:

```bash
qorechaind query bank total --denom uqor
```

## نصائح

:::caution
تحقق دائمًا مرتين من عناوين المستلِمين قبل إرسال الرموز. المعاملات في QoreChain لا يمكن التراجع عنها.
:::

:::tip

* استخدم العَلَم `--dry-run` لمحاكاة معاملة دون بثّها.
* استخدم `--gas auto` للسماح للعُقدة بتقدير الغاز المطلوب لمعاملتك.
* الحد الأدنى لرسوم التحويل القياسي هو **500 uqor**.

:::
