# تخزين xQORE

يغطي هذا الدليل آلية تخزين الحوكمة xQORE، التي تتيح لحاملي QOR قفل رموزهم للحصول على قوة حوكمة معزَّزة عبر نموذج إعادة توزيع PvP (لاعب ضد لاعب) يكافئ المشاركين على المدى الطويل.

:::note
تستخدم الأوامر أدناه شبكة الاختبار **`qorechain-diana`** (معرّف سلسلة EVM رقم **9800**). الشبكة الرئيسية (**`qorechain-vladi`**، معرّف سلسلة EVM رقم **9801**) أصبحت فعّالة منذ 7 يونيو 2026 وتعمل بإصدار السلسلة **v3.1.80** — استبدل معرّف السلسلة ونقاط النهاية الخاصة بالشبكة الرئيسية من صفحة **Connecting to Mainnet** عند التخزين على الشبكة الرئيسية.
:::

---

## نظرة عامة

xQORE هو رمز تخزين الحوكمة الخاص بـ QoreChain. عندما تقفل QOR، تتلقى xQORE بنسبة **1:1**. يوفّر الاحتفاظ بـ xQORE ميزة كبيرة في الحوكمة: تُحتسب رموز xQORE بـ **ضِعف الوزن** في صيغة قوة التصويت QDRW (انظر [الحوكمة](/user-guide/governance)).

```
VP = sqrt(staked + 2 * xQORE) * ReputationMultiplier(r)
```

هذا يعني أن قفل QOR في xQORE يضاعف فعليًا تأثيره على الحوكمة مقارنةً بالتخزين العادي وحده.

---

## قفل QOR مقابل xQORE

اقفل رموز QOR لسكّ xQORE بنسبة 1:1:

```bash
qorechaind tx xqore lock <amount>uqor \
  --from mykey \
  --chain-id qorechain-diana \
  --fees 500uqor
```

**مثال:** قفل 1,000 QOR:

```bash
qorechaind tx xqore lock 1000000000uqor \
  --from mykey \
  --chain-id qorechain-diana \
  --fees 500uqor
```

بعد هذه المعاملة، سيحتفظ حسابك بـ 1,000,000,000 uxqore (1,000 xQORE).

---

## فك قفل xQORE

احرق xQORE لاستعادة QOR. قد تُطبَّق **غرامة خروج** اعتمادًا على المدة التي ظلت فيها الرموز مقفلة:

```bash
qorechaind tx xqore unlock <amount>uqor \
  --from mykey \
  --chain-id qorechain-diana \
  --fees 500uqor
```

**مثال:** فك قفل 500 xQORE:

```bash
qorechaind tx xqore unlock 500000000uqor \
  --from mykey \
  --chain-id qorechain-diana \
  --fees 500uqor
```

## جدول غرامة الخروج

يترتب على السحب المبكر من xQORE غرامة. كلما طال احتفاظك، انخفضت الغرامة:

| مدة القفل      | غرامة الخروج |
| ------------------ | ------------ |
| أقل من 30 يومًا  | **50%**      |
| من 30 إلى 90 يومًا      | **35%**      |
| من 90 إلى 180 يومًا     | **15%**      |
| أكثر من 180 يومًا | **0%**       |

**مثال:** إذا قفلت 1,000 QOR وفككت القفل بعد 45 يومًا، فستتلقى 650 QOR (بعد تطبيق غرامة 35%). يُعاد توزيع الـ 350 QOR المتبقية على حاملي xQORE الآخرين عبر آلية إعادة التوزيع PvP.

---

## آلية إعادة التوزيع PvP

الغرامات المُحصَّلة من عمليات الخروج المبكر **لا تُحرق**. بدلًا من ذلك، يُعاد توزيعها تناسبيًا على جميع حاملي xQORE المتبقّين. وهذا يخلق ديناميكية "لاعب ضد لاعب" يستفيد فيها الحاملون الصبورون من نفاد صبر الآخرين.

كيف تعمل:

1. يقوم مستخدم بفك قفل xQORE قبل عتبة الـ 180 يومًا الخالية من الغرامة.
2. تُخصَم غرامة الخروج من QOR المُعاد إليه.
3. يُوزَّع مبلغ الغرامة تناسبيًا على جميع مراكز xQORE المتبقّية.
4. يزداد QOR القابل للمطالبة لكل xQORE لدى كل حامل متبقٍّ.

تحفّز هذه الآلية الالتزام طويل الأمد بالحوكمة وتكافئ الحاملين الذين يحافظون على مراكزهم.

## الاستعلام عن مركزك

تحقق من مركز xQORE الحالي ومدة القفل وغرامة الخروج المطبَّقة:

```bash
qorechaind query xqore position <address>
```

**مثال:**

```bash
qorechaind query xqore position qor1abc...xyz
```

**نموذج للمخرجات:**

```yaml
position:
  address: qor1abc...xyz
  locked_amount: "1000000000"
  xqore_balance: "1000000000"
  lock_timestamp: "2026-01-15T12:00:00Z"
  current_penalty_rate: "0.150000000000000000"
  accrued_rebase: "25000000"
```

## الوصول عبر JSON-RPC

بالنسبة للتطبيقات التي تتكامل مع QoreChain عبر JSON-RPC، يمكن الاستعلام عن مركز xQORE باستخدام:

```
qor_getXQOREPosition
```

**الطلب:**

```json
{
  "jsonrpc": "2.0",
  "method": "qor_getXQOREPosition",
  "params": ["qor1abc...xyz"],
  "id": 1
}
```

**الاستجابة:**

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "locked_amount": "1000000000",
    "xqore_balance": "1000000000",
    "lock_timestamp": "2026-01-15T12:00:00Z",
    "current_penalty_rate": "0.15",
    "accrued_rebase": "25000000"
  }
}
```

## نصائح

* اقفل QOR في xQORE قبل وقت كافٍ من عمليات التصويت المهمة في الحوكمة لتعظيم قوة تصويتك.
* عتبة الـ 180 يومًا للخروج الخالي من الغرامة تكافئ المشاركين الصبورين في الحوكمة.
* راقب تراكمات إعادة التوزيع PvP. كلما خرج الآخرون مبكرًا، نما مركزك في القيمة.
* xQORE غير قابل للتحويل. لا يمكن سكّه إلا بقفل QOR وحرقه إلا بفك القفل.
* ضع غرامة الخروج في اعتبارك بعناية قبل القفل. تحمل عمليات القفل قصيرة الأمد غرامات كبيرة.
