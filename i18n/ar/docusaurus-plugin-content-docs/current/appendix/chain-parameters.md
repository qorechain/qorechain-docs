# معاملات السلسلة

مرجع موحَّد لجميع معاملات الوحدات القابلة للتهيئة في تكوين منشأ QoreChain. تُجمَّع المعاملات حسب الوحدة ويمكن الاستعلام عنها أثناء التشغيل باستخدام `qorechaind query <module> params`.

:::note
القيم المعروضة هي الإعدادات الافتراضية المنشورة في منشأ الشبكة. تنطبق المعاملات على الشبكة الرئيسية **`qorechain-vladi`** (معرّف سلسلة EVM رقم **9801**) والشبكة التجريبية **`qorechain-diana`** (معرّف سلسلة EVM رقم **9800**) ما لم يُذكر خلاف ذلك.
:::

| المعامل                     | النوع  | القيمة الافتراضية      | الوصف                                                                  |
| --------------------------- | ------ | ---------------------- | ---------------------------------------------------------------------- |
| `hybrid_signature_mode`     | uint   | `2` (مطلوب)            | وضع الإلزام: 0=معطّل، 1=اختياري، 2=مطلوب (الافتراضي الحالي)            |

| `auto_register_enabled`     | bool   | `true`                 | التسجيل التلقائي لمفاتيح PQC عند أول معاملة هجينة                      |
| `migration_deadline_height` | uint64 | `0`                    | ارتفاع الكتلة الذي تُرفض بعده المفاتيح الكلاسيكية فقط (0=معطّل)        |
| `migration_grace_period`    | uint64 | `100000`               | عدد الكتل من التحذير قبل موعد ترحيل النهائي                            |

## وحدة السمعة (`x/reputation`)

| المعامل        | النوع  | القيمة الافتراضية | الوصف                                                |
| -------------- | ------ | ----------------- | --------------------------------------------------- |
| `alpha`        | string | `0.30`            | وزن درجة وقت التشغيل (S\_i) في معادلة السمعة         |
| `beta`         | string | `0.25`            | وزن درجة المشاركة (P\_i)                             |
| `gamma`        | string | `0.25`            | وزن درجة المجتمع (C\_i)                              |
| `delta`        | string | `0.20`            | وزن درجة مدة الخدمة (T\_i)                           |
| `decay_lambda` | string | `0.01`            | عامل التلاشي الزمني الأسي للدرجات التاريخية          |

معادلة السمعة: `R_i = alpha * S_i + beta * P_i + gamma * C_i + delta * T_i` مع تطبيق التلاشي الزمني الأسي لكل حقبة.

:::note
معاملات `x/inflation` أعلاه هي الإعدادات الافتراضية للآلية المنشورة. في إطار النموذج الاقتصادي الرسمي **tokenomics v2.1**، تتميز QoreChain بـ **عرض ثابت** مع **ميزانية إصدار محدودة (مجمَّع 590 مليون)** تموِّل مكافآت الرهان والنظام البيئي. قيم `initial_rate` / `rate_decay` / `min_rate` هي تفاصيل آلية تحكم كيفية جدولة الإصدارات ضمن تلك الميزانية المحدودة — وهي **ليست** تضخمًا مفتوحًا بالنسبة المئوية لإجمالي العرض.
:::

## وحدة تجريد الغاز (`x/gasabstraction`)

| المعامل           | النوع | القيمة الافتراضية | الوصف                                                |
| ----------------- | ----- | ----------------- | --------------------------------------------------- |
| `accepted_tokens` | array | (انظر أدناه)      | الرموز المقبولة لدفع الغاز مع معدلات التحويل         |

**الرموز المقبولة افتراضيًا:**

| فئة الرمز   | معدل التحويل    | الوصف                  |
| ----------- | --------------- | ---------------------- |
| `uqor`      | `1.0`           | رمز QOR الأصلي (1:1)    |
| `ibc/USDC`  | `1.0`           | USDC المجسور عبر IBC    |
| `ibc/ATOM`  | `10.0`          | ATOM المجسور عبر IBC    |

تمثّل معدلات التحويل عدد وحدات الغاز لكل وحدة رمز. المعدلات الأعلى تعني أن كل وحدة رمز تغطي قدرًا أكبر من الغاز.
