# الاتصال بشبكة الاختبار

انضم إلى شبكة QoreChain Diana للاختبار النشطة عبر تهيئة عقدتك بملف التكوين الجيني الصحيح والأقران وإعدادات الشبكة.

:::note
تغطّي هذه الصفحة شبكة **`qorechain-diana`** للاختبار (معرّف سلسلة EVM **9800**). الشبكة الرئيسية (**`qorechain-vladi`**، معرّف سلسلة EVM **9801**) نشطة منذ 7 يونيو 2026 ولها صفحة **الاتصال بالشبكة الرئيسية** المخصصة الخاصة بها مع تكوين جيني وأقران وتفاصيل اتصال منفصلة.
:::

---

## تنزيل التكوين الجيني

استبدل ملف التكوين الجيني المحلي بالتكوين الجيني الرسمي لشبكة الاختبار:

```bash
curl -o ~/.qorechaind/config/genesis.json \
  https://raw.githubusercontent.com/qorechain/qorechain-core/main/config/genesis.json
```

يحدّد هذا الملف الحالة الأولية لشبكة Diana للاختبار، بما في ذلك مجموعة المُحقّقين، وتخصيصات الرموز، ومعاملات الوحدات.

---

## تهيئة الأقران

عدّل تكوين عقدتك للاتصال بأقران شبكة الاختبار الموجودين.

افتح `~/.qorechaind/config/config.toml` واضبط حقل `persistent_peers`:

```toml
persistent_peers = "node-id@seed1.qorechain.io:26656,node-id@seed2.qorechain.io:26656"
```

راجع [مستودع QoreChain](https://github.com/qorechain/qorechain-core) للحصول على أحدث قائمة بالأقران.

### الإعدادات الموصى بها

قد ترغب أيضًا في تعديل ما يلي في `config.toml`:

```toml
[mempool]
size = 5000

[consensus]
timeout_propose = "3s"
timeout_commit = "5s"
```

هذه القيم مُضبطة لأوقات الكتل والإنتاجية في شبكة Diana للاختبار.

---

## تشغيل العقدة

شغّل عقدتك لبدء المزامنة مع الشبكة:

```bash
./qorechaind start
```

تتصل العقدة بالأقران وتبدأ بتنزيل الكتل من التكوين الجيني. يعتمد وقت المزامنة الأولي على ارتفاع السلسلة الحالي وسرعة شبكتك.

---

## التحقق من حالة المزامنة

تأكّد من أن عقدتك تلحق بأحدث كتلة:

```bash
curl localhost:26657/status | jq '.result.sync_info.catching_up'
```

* `true` — لا تزال العقدة تتزامن. انتظر حتى تلحق بالشبكة.
* `false` — العقدة متزامنة بالكامل وتعالج الكتل الجديدة.

يمكنك أيضًا التحقق من ارتفاع أحدث كتلة:

```bash
curl localhost:26657/status | jq '.result.sync_info.latest_block_height'
```

## المراقبة

تعرض QoreChain عدّة نقاط نهاية لمراقبة سلامة العقدة وأدائها.

### مقاييس Prometheus

تتوفر المقاييس الخام على:

```
http://localhost:26660/metrics
```

يمكن جمع هذه المقاييس بواسطة أي أداة جمع متوافقة مع Prometheus.

### لوحات معلومات Grafana

إذا كنت تشغّل عبر Docker Compose، فإن Grafana متاحة على:

```
http://localhost:3001
```

عند تسجيل الدخول لأول مرة، اضبط بيانات اعتمادك الخاصة عند المطالبة — لا تترك القيم الافتراضية كما هي. تعرض لوحات المعلومات المُهيّأة مسبقًا إنتاج الكتل، وإنتاجية المعاملات، واتصالات الأقران، واستخدام الموارد.

### فحص سلامة REST

توفّر واجهة REST API نقطة نهاية حالة سريعة:

```
http://localhost:1317
```

## مرجع المنافذ

| المنفذ    | البروتوكول  | الوصف                                        |
| ------- | --------- | -------------------------------------------------- |
| `26657` | TCP       | RPC — استعلام وبثّ المعاملات             |
| `26656` | TCP       | P2P — اتصالات شبكة الند للند           |
| `1317`  | HTTP      | REST API — استعلام حالة السلسلة عبر HTTP              |
| `9090`  | gRPC      | gRPC API — وصول برمجي إلى السلسلة               |
| `8545`  | HTTP      | EVM JSON-RPC — RPC متوافق مع إيثريوم (معرّف السلسلة `9800`) |
| `8546`  | WebSocket | EVM WebSocket — اشتراكات أحداث EVM في الوقت الفعلي  |
| `8899`  | HTTP      | SVM RPC — RPC متوافق مع سولانا                    |
| `26660` | HTTP      | نقطة نهاية مقاييس Prometheus                        |
