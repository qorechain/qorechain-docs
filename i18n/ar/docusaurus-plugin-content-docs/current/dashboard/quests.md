# المهام

**المهام** هي مهام موجَّهة ترشدك خلال استخدام QoreChain أثناء إنجازك لها. تجمع كل مهمة خطوة واحدة أو أكثر — مثل تمويل محفظتك، أو إجراء تحويل، أو تجربة عملية مبادلة — وتكافئك على إتمامها. تُعدّ المهام طريقة عملية لتعلُّم الشبكة ميزةً تلو الأخرى.

اربط محفظتك ليتم تتبُّع تقدُّمك — انظر [نظرة عامة والبدء](/dashboard/overview#connect-your-wallet).

## تصفُّح المهام

تُفتح صفحة المهام بملخص سريع لعدد المهام المتاحة، وكم منها قيد التنفيذ لديك، وكم منها أكملت. تتيح لك **علامات تبويب الفئات** — مثل الإعداد الأولي، والتداول، والتواصل الاجتماعي، والمجتمع، والتعلُّم، واليومية، والأسبوعية — تصفية القائمة.

تظهر كل مهمة على هيئة بطاقة تعرض:

- أيقونة وعنوانًا ووصفًا قصيرًا.
- شارة **مستوى الصعوبة** (على سبيل المثال سهل، أو متوسط، أو صعب).
- وقتًا تقديريًا للإتمام.
- **المكافأة** التي تحصل عليها.
- **الحالة**: مكتملة، أو قيد التنفيذ، أو بدأت، أو مقفلة.

تعرض المهمة المقفلة المهمة التي تحتاج إلى إنهائها أولًا قبل أن تُفتح.

## إتمام مهمة

1. افتح مهمة لرؤية صفحة تفاصيلها، مع القائمة الكاملة للمهام.
2. أنجز كل مهمة. قد تربطك المهمة بصفحة لوحة التحكم ذات الصلة لتنفيذ إجراء، أو تطلب منك تأكيد خطوة ليتسنى التحقق منها.
3. يتتبع شريط التقدُّم عدد المهام التي أكملتها.
4. عند إنجاز جميع المهام، اختر **Check Completion** للتحقق من المهمة والحصول على مكافأتها.

تُعاد بعض المهام إلى وضعها الأصلي وفق جدول يومي أو أسبوعي. بالنسبة لتلك المهام، تعرض صفحة التفاصيل وتيرة إعادة الضبط وسلسلتك المتتالية الحالية، لتتمكن من مواصلة سلسلتك بالعودة بانتظام.
