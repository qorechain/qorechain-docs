# Usare QCAIA su Discord

Su Discord, QCAIA funziona tramite **slash command**. Digita `/`, scegli un comando, inserisci la tua domanda e QCAIA risponde direttamente nel canale.

## Entra nel Discord

Entra nel server Discord ufficiale di QoreChain:

**[discord.gg/qorechain](https://discord.gg/qorechain)**

Una volta dentro, vai al canale **#general** per parlare con QCAIA.

## Slash command

QCAIA risponde a quattro slash command:

| Comando | Cosa fa |
| --- | --- |
| **`/ask <question>`** | Chiedi a QCAIA qualsiasi cosa su QoreChain. |
| **`/about`** | Scopri cos'è QoreChain e cosa può fare QCAIA. |
| **`/tokenomics`** | Ottieni un riepilogo del token QOR e della sua economia. |
| **`/contact`** | Trova i modi ufficiali per metterti in contatto con il team. |

## Fare una domanda

Usa **`/ask`** nel canale **#general**:

```
/ask How does QoreChain's bridge work?
```

QCAIA legge la tua domanda e risponde. Le risposte arrivano sotto forma di **embed di Discord** — schede di messaggio formattate che mantengono la risposta pulita e facile da leggere. Puoi porre domande di approfondimento allo stesso modo.

Poiché QCAIA è **multilingue**, puoi anche scrivere la tua domanda `/ask` in un'altra lingua; tradurrà, risponderà e replicherà. Consulta [Esempi](/qcaia/examples) per una dimostrazione.

## Suggerimenti

- Mantieni ogni `/ask` focalizzato su una sola domanda per ottenere la risposta più chiara.
- Usa **`/about`** se sei nuovo e vuoi un orientamento rapido.
- Usa **`/tokenomics`** per un riepilogo veloce di QOR prima di approfondire con `/ask`.
- QCAIA è in **beta** — se una risposta non è del tutto corretta, prova a riformulare la domanda.

Per l'esperienza basata sulle menzioni su Telegram, consulta invece [Usare QCAIA su Telegram](/qcaia/telegram).
