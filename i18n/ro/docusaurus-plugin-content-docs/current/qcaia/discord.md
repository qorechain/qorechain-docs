# Folosirea QCAIA pe Discord

Pe Discord, QCAIA funcționează prin **comenzi slash**. Tastează `/`, alege o comandă, completează-ți întrebarea, iar QCAIA răspunde chiar în canal.

## Alătură-te Discord-ului

Alătură-te serverului Discord oficial QoreChain:

**[discord.gg/qorechain](https://discord.gg/qorechain)**

Odată intrat, mergi la canalul **#general** pentru a vorbi cu QCAIA.

## Comenzi slash

QCAIA răspunde la patru comenzi slash:

| Comandă | Ce face |
| --- | --- |
| **`/ask <question>`** | Întreabă QCAIA orice despre QoreChain. |
| **`/about`** | Află ce este QoreChain și ce poate face QCAIA. |
| **`/tokenomics`** | Obține un rezumat al tokenului QOR și al economiei sale. |
| **`/contact`** | Găsește modalități oficiale de a lua legătura cu echipa. |

## Punerea unei întrebări

Folosește **`/ask`** în canalul **#general**:

```
/ask How does QoreChain's bridge work?
```

QCAIA îți citește întrebarea și răspunde. Răspunsurile vin sub formă de **embed-uri Discord** — carduri de mesaj formatate care păstrează răspunsul curat și ușor de citit. Poți pune întrebări de continuare în același mod.

Deoarece QCAIA este **multilingv**, îți poți scrie întrebarea `/ask` și în altă limbă; va traduce, va răspunde și va replica. Vezi [Exemple](/qcaia/examples) pentru o demonstrație.

## Sfaturi

- Menține fiecare `/ask` concentrat pe o singură întrebare pentru cel mai clar răspuns.
- Folosește **`/about`** dacă ești nou și vrei o orientare rapidă.
- Folosește **`/tokenomics`** pentru un rezumat rapid al QOR înainte de a aprofunda cu `/ask`.
- QCAIA este în **beta** — dacă un răspuns nu este tocmai corect, încearcă să-ți reformulezi întrebarea.

Pentru experiența Telegram bazată pe menționare, vezi [Folosirea QCAIA pe Telegram](/qcaia/telegram).
