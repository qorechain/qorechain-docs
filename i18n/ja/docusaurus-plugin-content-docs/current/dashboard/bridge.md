# ブリッジ

**ブリッジ**を使うと、1つの画面から QoreChain と他のチェーンの間で資産を移動できます。すべてのブリッジ操作はポスト量子暗号によって保護されています。クロスチェーン転送の設計については、[ブリッジアーキテクチャ](/architecture/bridge-architecture)を参照してください。

:::caution 限定提供ステータス
クロスチェーンブリッジは現在テストネット上にあり、段階的に展開されています。まだ本番のメインネットシステムではありません。利用可能なルートは確実に稼働している接続ではなく、進行中の作業として扱い、まずはテストネットで開始してください。
:::

ブリッジを使用するにはウォレットを接続してください。[概要とはじめに](/dashboard/overview#connect-your-wallet)を参照してください。

## 資産をブリッジする方法

1. 上部のセレクターで**ソース**チェーンとトークンを選択します。セレクターにはトークン、そのネットワーク、および残高が表示されます。
2. 下部のセレクターで**宛先**チェーンとトークンを選択します。
3. 転送する金額を入力します。受け取る金額は宛先側に表示されます。
4. 自分以外のアドレスに資産を送る場合は、**別の宛先に送信**をオンにして受取人を入力します。
5. 下部に表示される**手数料**と決済までの**見積もり時間**を確認します。
6. ウォレットで転送を承認します。

2つのセレクターの間にあるスワップコントロールを使うと、1タップでソースと宛先を入れ替えられます。

## ヒント

- 送信前に両方のチェーンと宛先アドレスを確認してください。クロスチェーン転送は取り消すことができません。
- 決済時間はルートによって異なります。チェーンと金額を変更すると見積もりが更新されます。
- チェーン間で転送がどのように検証されるかの背景については、[資産のブリッジ](/user-guide/bridging-assets)と[ブリッジアーキテクチャ](/architecture/bridge-architecture)を参照してください。
