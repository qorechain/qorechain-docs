# エクスプローラー

**エクスプローラー**は、ダッシュボードからチェーンを覗くための窓口です。ブロック、トランザクション、アドレス、バリデーターを検索したり、ネットワークの活動をリアルタイムで監視したりするのに使用します。エクスプローラーは読み取り専用であり、閲覧するのにウォレット接続は必要ありません。

## 概要ページ

ダッシュボードから**エクスプローラー**を開くと、ネットワークのライブスナップショットが表示されます。

- **ネットワークステータス** — チェーン ID、現在のステータス、および量子安全インジケーター。
- **ブロック活動** — 最新のブロック高、平均ブロック時間、本日生成されたブロック数。
- **供給量** — ボンドされた QOR の総量、ボンド比率、循環供給量。
- **主要統計** — 総トランザクション数、アクティブおよび総バリデーター数、総アドレス数。
- **最新ブロック** — 各ブロックの高さ、時刻、トランザクション数、プロポーザーを含むライブ一覧。
- **最新トランザクション** — 各トランザクションのハッシュ、タイプ、ブロック、金額、送信者を含むライブ一覧。

任意のブロックまたはトランザクションの行をクリックすると、その詳細ページが開きます。各一覧の更新コントロールで最新のエントリを取得できます。

## 検索

エクスプローラー上部の検索ボックスは以下のいずれかを受け付け、自動的に適切なページへ案内します。

- **アドレス**（`qor1...`）
- **トランザクションハッシュ**
- **ブロック高**（数値）

## トランザクションの詳細

トランザクションページには、ハッシュ、ステータス、金額、送信者と受取人（どちらもクリック可能）、手数料、ブロック高、トランザクションタイプ、および存在する場合はメモが表示されます。ハッシュをコピーしたり、より詳しく確認するためにトランザクション全体の生データ表示を切り替えたりできます。

## ブロックの詳細

ブロックページには、高さ、タイムスタンプ、プロポーザー、ハッシュ、トランザクション数、使用ガス、それに含まれるトランザクションの一覧が、コンセンサスおよびポスト量子署名情報とともに表示されます。前へ・次へのコントロールで、ブロックごとにチェーンをたどることができます。

## アドレスの詳細

アドレスページには、スキャン可能な QR コード付きのアドレス、その QOR 残高、トランザクション数、送受信の合計が表示されます。その下には、そのアドレスの完全なトランザクション履歴（転送、スワップ、フォーセット請求など）が、それぞれの金額、時刻、ステータスとともに表示されます。アドレスのコピー、QR コードのダウンロード、任意のトランザクションを開いて詳細を確認することができます。

## バリデーター {#validators}

バリデータービューには、アクティブなバリデーター数、ボンドされた QOR の総量、コンセンサスの健全性のサマリーカードとともに、ネットワークのバリデーターが一覧表示されます。テーブルには各バリデーターのランク、モニカー、投票権、手数料、ステータス（例：アクティブまたは jailed）、およびポスト量子インジケーターが表示されます。検索ボックスでバリデーター名またはアドレスによってフィルターできます。バリデーターに委任するには、[ステーキングとバリデーター](/dashboard/staking-and-validators)を参照してください。
