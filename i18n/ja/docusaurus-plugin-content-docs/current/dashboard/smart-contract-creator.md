# スマートコントラクトクリエーター

**スマートコントラクトクリエーター**は、**QCAI** を活用して、平易な言葉による説明からスマートコントラクトを生成します。やりたいことを記述し、対象のブロックチェーンを選ぶと、QCAI がコントラクトを書いてくれます。AI ツールでは **17 のブロックチェーン**に対応しているため、構築先のエコシステムを対象にできます。

ウォレットを接続すると、生成したコントラクトの保存と管理ができます。詳しくは[概要とはじめに](/dashboard/overview#connect-your-wallet)をご覧ください。

## コントラクトを生成する

1. **コントラクトを記述する。** プロンプト欄に、コントラクトが何をすべきかを記述します。例えば、固定供給量のトークン、NFT コレクション、ベスティングスケジュールなどです。具体的であるほど、より良い結果が得られます。
2. **ブロックチェーンを選ぶ。** 対応ブロックチェーンの中から対象を選択します。選んだものに対応するコントラクト言語とカテゴリが、セレクタの横に表示されます。
3. **コントラクトタイプを選ぶ**（任意）。トークン、NFT、ガバナンスコントラクトなどの開始テンプレートを選んで、生成をガイドします。
4. **生成する。** **Generate** を選択します。QCAI がコントラクトを生成する間、進捗インジケーターでステータスが表示されます。

## 結果を確認する

生成が完了すると、ダッシュボードにシンタックスハイライト付きのビューでコントラクトが表示され、コントラクト名、ブロックチェーン、言語、ファイルサイズ、生成日時などの詳細も併せて示されます。使用したプロンプトも、参照用として結果とともに表示されます。

ここから次のことができます。

- コントラクトコードをクリップボードに**コピー**する。
- 選んだブロックチェーンに適した形式のファイルとしてコントラクトを**ダウンロード**する。
- コントラクトを**編集**してさらに洗練させる。

## 共有と再利用

生成された各コントラクトには、開いたり共有したりできる専用ページがあります。自分が所有していないコントラクトを開いた場合は、それを**フォーク**して自分用のコピーを作り、そこから続けることができます。

:::tip 必ず確認とテストを
QCAI が生成したコードは強力な出発点ですが、確認の代わりにはなりません。コントラクトを読み、[テストネット](/getting-started/connecting-to-testnet)でテストし、価値のあるものをデプロイする前に[コントラクト監査ツール](/dashboard/contract-auditor)に通してください。
:::
