# ダッシュボードの概要とはじめに

**[dashboard.qorechain.io](https://dashboard.qorechain.io)** にある QoreChain ダッシュボードは、ブラウザから QoreChain を利用するための公式ウェブアプリです。1つの場所から、チェーンの探索、ウォレットの管理、トークンのスワップ、チェーン間での資産移動、スマートコントラクトの生成と監査、バリデーターへのステーキング、テストネットトークンの請求、クエストの完了、ネットワークのツーリングへのアクセスが行えます。

このセクションの内容はすべてユーザー向けのハウツーです。各ページが何をするものか、そしてどのように使うかを説明します。インストールは不要で、ダッシュボードはすべてブラウザ内で動作します。

## できること

| エリア | 用途 |
| --- | --- |
| **[エクスプローラー](/dashboard/explorer)** | ブロック、トランザクション、アドレス、バリデーターを閲覧します。 |
| **[ウォレット](/dashboard/wallet)** | 残高の表示、QOR の送受信、アドレスの管理を行います。 |
| **[トレード](/dashboard/trade)** | オンチェーン AMM でトークンをスワップし、流動性を提供します。 |
| **[ブリッジ](/dashboard/bridge)** | QoreChain と他のチェーンの間で資産を移動します。 |
| **[スマートコントラクトクリエイター](/dashboard/smart-contract-creator)** | サポートされている 17 のブロックチェーンにわたって **QCAI** でスマートコントラクトを生成します。 |
| **[コントラクト監査ツール](/dashboard/contract-auditor)** | スマートコントラクトに対して **QCAI** セキュリティ分析を実行します。 |
| **[ステーキングとバリデーター](/dashboard/staking-and-validators)** | バリデーターを確認し、QOR を委任します。 |
| **[フォーセット](/dashboard/faucet)** | テストネットでテストトークンをリクエストします。 |
| **[クエスト](/dashboard/quests)** | ガイド付きのタスクを完了してネットワークを学びます。 |
| **[ツールハブ](/dashboard/tools-hub)** | ノード、ロールアップ、SDK、ライセンスのツーリングにアクセスします。 |

## ウォレットを接続する {#connect-your-wallet}

オンチェーンの状態を変更するほとんどのアクション（トークンの送信、スワップ、ステーキング、ブリッジ）には、接続済みのウォレットが必要です。

1. [dashboard.qorechain.io](https://dashboard.qorechain.io) を開きます。
2. **ウォレットを接続**を選択します。
3. ウォレットで接続を承認します。

接続すると、ダッシュボードのヘッダーにアドレス（短縮形）が表示され、署名が必要なアクションがアンロックされます。エクスプローラーなどの読み取り専用ページは接続なしで動作します。

QoreChain アカウントは `qor` の bech32 プレフィックスを使用するため、接続済みのアドレスは `qor1...` のように表示されます。アカウントは量子安全暗号で保護されています。初回のセットアップガイダンスについては、[ウォレットのセットアップ](/getting-started/wallet-setup)を参照してください。

## ネットワークを選択する

ダッシュボードは 2 つのネットワークに対して動作します。ヘッダーには現在接続しているネットワークが表示されます。

| ネットワーク | チェーン ID | 使用するタイミング |
| --- | --- | --- |
| **メインネット** | `qorechain-vladi` | 実際の価値と本番利用のためのライブネットワーク。 |
| **テストネット** | `qorechain-diana` | テスト用の無料環境で、テストトークン用の[フォーセット](/dashboard/faucet)を備えています。 |

ネイティブトークンは **QOR**（基本単位は `uqor`、1 QOR = 10^6 uqor）です。初めての場合は、テストネットで開始し、フォーセットからトークンを請求し、メインネットに移行する前に最初の転送を試してください。

:::tip QoreChain が初めてですか？
[テストネットへの接続](/getting-started/connecting-to-testnet)と[最初のトランザクション](/getting-started/first-transaction)に従ってすぐに実践し、それからダッシュボードの残りの部分を探索しに戻ってきてください。
:::
