# ウォレット

**ウォレット**ページでは、残高の確認、QOR の送受信、アドレスの管理を行えます。QoreChain のアカウントは耐量子暗号で保護されており、すべてのアドレスは `qor` bech32 プレフィックス（`qor1...`）を使用します。

まずウォレットを接続してください。詳しくは[概要とはじめに](/dashboard/overview#connect-your-wallet)を参照してください。

## このページに表示される内容

- ウォレットのラベルと現在アクティブなアドレス（短縮形）、およびワンクリックのコピーボタン。
- QOR 建ての**合計残高**。
- 耐量子暗号化と接続中のネットワークを示すセキュリティパネル。
- 更新コントロール付きの最終更新インジケーター。
- 保有資産と取引履歴を表示する**アセット**タブと**アクティビティ**タブ。

更新コントロールはいつでも使用でき、現在の残高と最新のアクティビティをチェーンから取得します。

## QOR を送信する

1. **送信**を選択します。
2. 受取人のアドレス（`qor1...`）を入力します。
3. 金額と、任意でメモを入力します。
4. 詳細と推定手数料を確認し、確定します。

受取人を入力すると、保存済みの連絡先と最近使用したアドレスが候補として表示され、入力ミスを防ぐのに役立ちます。送金が送信されると、トランザクションハッシュ付きの確認が表示され、[エクスプローラー](/dashboard/explorer)で開くことができます。

:::caution アドレスを再確認してください
ブロックチェーンの送金は取り消せません。送信前に必ず受取人のアドレスを確認してください。
:::

## QOR を受信する

1. **受信**を選択します。
2. 送信者にアドレスまたはその QR コードを共有するか、ワンクリックでアドレスをコピーします。
3. 任意で要求金額とメモを入力すると、支払いリンクとダウンロード可能な QR コードを生成できます。

## ウォレットを管理する

**マイウォレット**を選択するとアドレス一覧が開きます。そこでウォレットの切り替え、新しいウォレットの作成、既存ウォレットのインポート、不要になったウォレットの削除が行えます。アクティブなウォレットは、ダッシュボード全体での送信、スワップ、ステーキング、その他の署名付きアクションに使用されます。
