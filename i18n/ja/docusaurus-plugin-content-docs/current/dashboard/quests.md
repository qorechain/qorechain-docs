# クエスト

**クエスト**は、完了しながら QoreChain の使い方を案内してくれるガイド付きタスクです。各クエストは、ウォレットへの資金供給、送金、スワップのお試しといった 1 つ以上のステップをまとめており、完了すると報酬が得られます。クエストは、ネットワークの機能を 1 つずつ実践的に学ぶための方法です。

進捗が記録されるようウォレットを接続してください。詳しくは[概要とはじめに](/dashboard/overview#connect-your-wallet)をご覧ください。

## クエストを閲覧する

クエストページを開くと、利用可能なクエスト数、進行中のクエスト数、完了済みのクエスト数の概要がすぐに表示されます。**カテゴリタブ**（オンボーディング、トレーディング、ソーシャル、コミュニティ、学習、デイリー、ウィークリーなど）でリストを絞り込めます。

各クエストはカードとして表示され、次の内容が示されます。

- アイコン、タイトル、短い説明。
- **難易度**バッジ（例：easy、medium、hard）。
- 完了までの推定時間。
- 獲得できる**報酬**。
- **ステータス**：完了、進行中、開始済み、ロック中。

ロックされたクエストには、解除する前に先に完了する必要があるクエストが表示されます。

## クエストを完了する

1. クエストを開くと、タスクの全リストが載った詳細ページが表示されます。
2. 各タスクを進めます。タスクによっては、アクションを実行するために関連するダッシュボードページへのリンクが用意されていたり、検証できるようステップの確認を求められたりします。
3. 進捗バーで、完了したタスク数が追跡されます。
4. すべてのタスクが終わったら、**Check Completion** を選択してクエストを検証し、報酬を受け取ります。

一部のクエストは、デイリーまたはウィークリーのスケジュールでリセットされます。それらのクエストでは、詳細ページにリセットの周期と現在のストリークが表示されるため、定期的に戻ってくることで連続記録を維持できます。
