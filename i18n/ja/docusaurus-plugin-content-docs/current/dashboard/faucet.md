# フォーセット

**フォーセット**は無料のテストトークンを提供するため、価値のあるものを消費せずにダッシュボードを試すことができます。これは**テストネット専用**のツールです。テストネット（`qorechain-diana`）で使用し、転送、スワップ、ステーキング、コントラクトのデプロイをテストする前にアドレスに資金を入れてください。

## テストトークンをリクエストする

1. **テストネット**に接続し、**フォーセット**を開きます。
2. 資金を入れるアドレス（`qor1...`）を入力します。ウォレットが接続されている場合は、**自分のアドレスを使用**を選択すると自動的に入力されます。フォームは続行する前にアドレスが有効であることを確認します。
3. リクエストボタンを選択します。資金は数秒以内に処理されます。

リクエストが成功すると、確認カードに送信された金額とトランザクションハッシュが、コピーボタンおよび[エクスプローラー](/dashboard/explorer)でトランザクションを開くためのリンクとともに表示されます。

## 制限

各アドレスは、クールダウン期間ごとに 1 回フォーセットにリクエストできます。ページには 1 回のリクエストあたりの正確な金額と、再度請求できるまでのクールダウンが表示されます。早すぎるタイミングで再度リクエストした場合、フォーセットは次に請求可能になる時期を知らせます。

## テストトークンの使い道

テストトークンを使うと、テストネット上でネットワークをエンドツーエンドで試すことができます。

- [ウォレット](/dashboard/wallet)ページで送受信する。
- AMM で[スワップ](/dashboard/trade)を試す。
- バリデーターに[委任](/dashboard/staking-and-validators)する。
- メインネットに移行する前にコントラクトをデプロイしてテストする。

:::note テスト用の価値のみ
フォーセットトークンはテストネット上に存在し、実際の価値はありません。本番利用の準備ができたら、メインネット（`qorechain-vladi`）に切り替えてください。
:::
