# トレード（DEX）

**トレード**ページは、ダッシュボードの分散型取引所です。QoreChain のネイティブなオンチェーン自動マーケットメーカー（AMM）上で、トークンのスワップや流動性の提供を直接行えます。オフチェーンのオーダーブックも外部のスマートコントラクトも不要です。スワップと流動性の操作は、プロトコルレベルで決済されます。その背後にある設計については、[AMM とオンチェーン流動性](/architecture/amm)をご覧ください。

トレードするにはウォレットを接続してください。詳しくは[概要とはじめに](/dashboard/overview#connect-your-wallet)をご覧ください。

このページには 4 つのタブがあります。**Swap**、**Pools**、**Add Liquidity**、**My Positions** です。

## Swap

あるトークンを別のトークンに交換します。

1. 支払いに使うトークンを選び、金額を入力します。
2. 受け取りたいトークンを選びます。出力額は自動的に見積もられます。
3. 必要に応じて、**スリッページ許容度**（許容する最大の価格変動。デフォルトは 0.5%）を調整します。
4. **Swap** を選択して確定します。

スワップ履歴パネルには、過去のスワップがトークン、金額、レート、時刻、ステータスとともに一覧表示されます。

:::tip スリッページ
スリッページ許容度を高くすると、値動きの速い市場でもスワップが成立しやすくなりますが、受け取る価格は不利になる場合があります。デフォルトはほとんどのペアに適しています。
:::

## Pools

利用可能な流動性プールを閲覧します。各プールカードには、取引ペア、流動性の合計、24 時間の出来高、APR、プロバイダー数が表示されます。検索ボックスでトークンシンボルによりプールを絞り込めます。

## Add Liquidity

流動性を提供してスワップ手数料の分け前を獲得します。

1. ペアにする 2 つのトークンを選びます（一方はデフォルトで QOR）。
2. 最初のトークンの金額を入力します。2 つ目の金額は、現在のプール比率に合わせて自動的に入力されます。
3. プールに占める見込みの持分を確認し、**Add Liquidity** を選択して確定します。

ポジションを表す流動性プロバイダー（LP）トークンを受け取ります。

## My Positions

保有している流動性ポジションを表示します。各エントリには、トークンペア、各トークンの金額、プールに占める持分、獲得した手数料、APR が表示されます。ポジションの **Remove Liquidity** を選択すると、受け取れるトークンをプレビューし、自分の持分を引き出せます。
