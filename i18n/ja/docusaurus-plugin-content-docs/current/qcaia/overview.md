# QCAIA コミュニティボットの概要

**QCAIA** は公式の **QoreChain AI アシスタント** であり、QoreChain に関する質問に平易な言葉で答えるコミュニティ Q&A ボットです。QoreChain の AI エンジンである **QCAI** によって動作しており、現在 **ベータ版** です。

QCAIA はコミュニティがすでに集まっている場所に存在します。**Telegram** と **Discord** で利用できます。質問すると、QoreChain の公式ドキュメントに基づいた、簡潔でブランディングに準拠した回答が返ってきます。

## QCAIA が答えられること

QCAIA は、新規参加者やコミュニティメンバーが素早く理解できるよう支援するために作られています。次のような質問に答えられます:

- **技術** — QoreChain がどのように動作するかの全体像、そのコンセンサス、セキュリティ、そして [PRISM コンセンサスエンジン](/architecture/prism-consensus-engine)。
- **トークノミクス** — QOR トークン、手数料の分配、供給量。
- **チームとプロジェクト** — QoreChain の背後にいる人々と、プロジェクトの方向性。
- **エコシステム** — ブリッジ、AMM、ロールアップ、ダッシュボード、ネットワークのその他の部分。

質問が QCAIA の知っている範囲外であれば、推測するのではなく、その旨を伝えます。QCAIA はコミュニティのヘルパーであり、金融や投資のアドバイスではありません。

## 多言語サポート

QCAIA は多くの言語の質問を理解します。英語以外の言語で質問した場合、QCAIA は自動的にあなたの質問を翻訳し、回答して返信します。そのため、自分の言語を使いながらも明確な回答を得ることができます。英語以外の実例については [使用例](/qcaia/examples) を参照してください。

## ブランディング準拠フィルター

QCAIA が生成するすべての応答は、あなたが目にする前に **ブランディング準拠フィルター** を通過します。これにより、回答が QoreChain の公式用語と一貫し、常にブランドに沿ったものに保たれます。

## ベータ版のステータス

QCAIA は **ベータ版** です。継続的に改善されており、まれに答えを持っていなかったり、言い換えをお願いしたりすることがあります。コミュニティからのフィードバックが、その成長を直接的に形作っています。
