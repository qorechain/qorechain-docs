# Discord で QCAIA を使う

Discord では、QCAIA は**スラッシュコマンド**を通じて動作します。`/` を入力してコマンドを選び、質問を入力すると、QCAIA がそのチャンネル内で直接返信します。

## Discord に参加する

公式の QoreChain Discord サーバーに参加してください。

**[discord.gg/qorechain](https://discord.gg/qorechain)**

参加したら、**#general** チャンネルに移動して QCAIA と話してください。

## スラッシュコマンド

QCAIA は4つのスラッシュコマンドに応答します。

| コマンド | 機能 |
| --- | --- |
| **`/ask <question>`** | QoreChain について QCAIA に何でも質問できます。 |
| **`/about`** | QoreChain とは何か、QCAIA に何ができるかを学べます。 |
| **`/tokenomics`** | QOR トークンとその経済性の概要を取得します。 |
| **`/contact`** | チームに連絡する公式な方法を確認できます。 |

## 質問する

**#general** チャンネルで **`/ask`** を使います。

```
/ask How does QoreChain's bridge work?
```

QCAIA はあなたの質問を読み取り、返信します。応答は **Discord エンベッド** として返ってきます — 回答を整然と読みやすく保つ、フォーマットされたメッセージカードです。同じ方法でフォローアップの質問もできます。

QCAIA は**多言語対応**なので、`/ask` の質問を別の言語で書くこともできます。翻訳して回答し、返信します。デモンストレーションについては[例](/qcaia/examples)を参照してください。

## ヒント

- 最も明確な回答を得るには、各 `/ask` を1つの質問に絞ってください。
- 初めてで手早く全体像を知りたい場合は **`/about`** を使ってください。
- `/ask` でさらに掘り下げる前に、手早い QOR の概要を得るには **`/tokenomics`** を使ってください。
- QCAIA は**ベータ版**です — 回答が今ひとつ正確でない場合は、質問を言い換えてみてください。

代わりにメンションベースの Telegram 体験については、[Telegram で QCAIA を使う](/qcaia/telegram)を参照してください。
