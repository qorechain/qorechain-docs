# Telegram での QCAIA の使用

Telegram では、QCAIA は **メンションベース** です。スラッシュコマンドはありません。ボット（または QoreChain）にメンションしながら、自然に質問するだけで返信が返ってきます。

## グループへの参加

公式の QoreChain コミュニティグループに参加してください:

**[t.me/QoreChainCommunity](https://t.me/QoreChainCommunity)**

QCAIA は **グループ専用** で、Telegram の **フォーラムトピック** をサポートしているため、グループ内の整理されたトピックスレッドの中で動作します。

## QCAIA への話しかけ方

QCAIA は、次のいずれかを行ったときに返信します:

1. **直接メンションする** — メッセージに **`@QCAIA_Bot`** を含めます。
2. **そのメッセージに返信する** — QCAIA が送信したメッセージに返信またはフォローアップすると、応答します。
3. **「QoreChain」とメンションする** — **General** トピックでメッセージに「QoreChain」を含めると、QCAIA がそれを拾って回答します。

いずれの場合も、いつもどおりに質問を書くだけです。

### 例

ボットに直接メンションする:

```
@QCAIA_Bot How does QoreChain's bridge work?
```

**General** トピックで QoreChain にメンションする:

```
What is the QOR fee distribution on QoreChain?
```

QCAIA のメッセージに返信すると、メンションなしで会話を続けられます。

## ヒント

- **General** 以外のトピックスレッドでは、QCAIA が質問を確実に認識できるよう、**`@QCAIA_Bot`** にメンションする（または返信する）ようにしてください。
- QCAIA は **多言語対応** です。自分の言語で質問すると、翻訳し、回答して返信します。[使用例](/qcaia/examples) を参照してください。
- 最も明確な回答を得るには、各質問を焦点を絞ったものにしてください。
- QCAIA は **ベータ版** です。回答が適切でない場合は、言い換えてもう一度試してください。

スラッシュコマンドのほうがよいですか？ [Discord での QCAIA の使用](/qcaia/discord) を参照してください。
