# Misiones

Las **Misiones** son tareas guiadas que te enseñan a usar QoreChain a medida que las completas. Cada misión agrupa uno o más pasos —como financiar tu billetera, hacer una transferencia o probar un intercambio— y te recompensa por terminarla. Las misiones son una forma práctica de aprender la red característica por característica.

Conecta tu billetera para que se registre tu progreso —consulta [Resumen y primeros pasos](/dashboard/overview#connect-your-wallet).

## Explorar misiones

La página de Misiones se abre con un resumen rápido de cuántas misiones hay disponibles, cuántas tienes en curso y cuántas has completado. Las **pestañas de categoría** —como incorporación, trading, social, comunidad, aprendizaje, diarias y semanales— te permiten filtrar la lista.

Cada misión aparece como una tarjeta que muestra:

- Un icono, un título y una descripción breve.
- Una insignia de **dificultad** (por ejemplo, fácil, media o difícil).
- Un tiempo estimado para completarla.
- La **recompensa** que obtienes.
- Un **estado**: completada, en curso, iniciada o bloqueada.

Una misión bloqueada indica la misión que necesitas terminar primero antes de que se desbloquee.

## Completar una misión

1. Abre una misión para ver su página de detalle, con la lista completa de tareas.
2. Avanza por cada tarea. Una tarea puede enlazar a la página correspondiente del Panel para realizar una acción, o pedirte que confirmes un paso para poder verificarlo.
3. Una barra de progreso registra cuántas tareas has completado.
4. Cuando todas las tareas estén hechas, selecciona **Check Completion** para verificar la misión y recibir su recompensa.

Algunas misiones se reinician con una periodicidad diaria o semanal. Para esas, la página de detalle muestra la cadencia de reinicio y tu racha actual, de modo que puedas mantener la racha volviendo con regularidad.
