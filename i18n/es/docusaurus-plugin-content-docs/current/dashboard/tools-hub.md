# Centro de herramientas

La página de **Herramientas** reúne en un solo lugar las herramientas para operadores y desarrolladores de QoreChain, organizadas en pestañas. Desde aquí puedes registrar infraestructura, desplegar un rollup, acceder al SDK, solicitar convertirte en validador y adquirir las licencias que estos roles requieren. Cada sección se resume a continuación, indicando adónde lleva para la documentación completa.

Conecta tu billetera para usar las herramientas que registran infraestructura o envían solicitudes —consulta [Resumen y primeros pasos](/dashboard/overview#connect-your-wallet).

## Nodo ligero

Registra un nodo ligero para que pueda unirse a la red y generar recompensas. La sección ofrece un formulario de registro para tu dirección de operador, moniker y endpoint público, y confirma el compromiso de stake requerido para participar. Primero se necesita una licencia de nodo ligero activa; la sección enlaza a **Buy License** si no tienes una. Para una visión completa, consulta [Resumen del nodo ligero](/light-node/overview) y [Registro y licencias](/light-node/registration-and-licensing).

## Registro de nodo

Registra un nodo validador. El formulario recopila tu dirección de validador, moniker, tasa de comisión, endpoint público opcional y las redes que quieres validar, y confirma el compromiso de auto-stake. Se necesita una licencia de validador activa; la sección enlaza a **Buy License** cuando es necesario. Consulta [Ejecutar un validador](/developer-guide/running-a-validator) y [Staking y validadores](/dashboard/staking-and-validators).

## Rollups

Despliega tu propio rollup impulsado por QoreChain. El formulario de configuración te permite nombrar el rollup y elegir su máquina virtual (EVM, CosmWasm o SVM), capa de disponibilidad de datos, token de gas, modelo de secuenciador y destino de liquidación. Después de enviarlo, el rollup se aprovisiona tras una revisión antes de ponerse en marcha. Consulta [Resumen de rollups](/rollups/overview) y [Desplegar un rollup](/rollups/deploying-a-rollup).

## SDK

Un centro de inicio rápido y referencia para construir sobre QoreChain mediante código. La sección muestra los pasos de instalación y fragmentos listos para copiar para conectarse, derivar cuentas en los tres entornos de ejecución, leer estado, enviar transferencias y firmar de forma cuántica-segura, además de una tabla de paquetes por lenguaje y enlaces al repositorio, ejemplos y explorador. Consulta [Resumen del SDK de QoreChain](/sdk/overview) e [Instalación](/sdk/install).

## Solicitud de validador {#validator-application}

Solicita convertirte en Validador Genesis. El formulario recopila los datos de tu entidad y contacto, el nivel deseado, la región de infraestructura, información de hardware y motivación, con una declaración de cumplimiento. Después de enviarlo, se muestra el estado de la solicitud —en revisión, aprobada o no aprobada con un motivo— y, una vez aprobada, puedes proceder a adquirir una licencia de validador. Consulta [Ejecutar un validador](/developer-guide/running-a-validator).

## Comprar licencia

Adquiere las licencias necesarias para ejecutar la infraestructura de la red, incluidas las licencias de nodo ligero y de validador, además de complementos opcionales entre redes. Cada opción muestra lo que incluye y el rol que desbloquea; las licencias de validador quedan disponibles una vez que tu [Solicitud de validador](#validator-application) es aprobada. Para saber cómo funcionan las licencias en toda la red, consulta [Licencias de cadena](/architecture/chain-licensing).
